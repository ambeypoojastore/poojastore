📖 Setup Guide

Step 1 — Razorpay (Payment Gateway)

  1. Go to razorpay.com → Sign Up (free account)
  2. Complete KYC with Aadhaar + PAN + Bank account details
  3. Go to Settings → API Keys → Generate Key
  4. Copy the Key ID (starts with rzp_live_...)
  5. Paste in Settings → Payment section above
💰 Cost: ₹0 setup fee. Only 2% charged per successful transaction.

Step 2 — Google Sheets Database

  1. Create a new Google Sheet named AmbeyStoerDB
  2. Create 2 tabs: Orders and Feedback
  3. Go to Extensions → Apps Script → paste code below
  4. Deploy → New Deployment → Web App → Access: Anyone
  5. Copy Web App URL → paste in Settings → Payment section
const SS = SpreadsheetApp.getActiveSpreadsheet();
function doPost(e) {
  const d = JSON.parse(e.postData.contents);
  if(d.action==='saveOrder') {
    SS.getSheetByName('Orders').appendRow([
      d.id, d.customer.name, d.customer.phone,
      d.customer.addr, d.items.map(i=>i.name+'×'+i.qty).join(', '),
      d.total, d.paymentId, 'Placed', new Date()]);
  }
  if(d.action==='saveFeedback') {
    SS.getSheetByName('Feedback').appendRow([
      d.name, d.rating, d.message, d.orderId, new Date()]);
  }
  if(d.action==='updateStatus') {
    const rows=SS.getSheetByName('Orders').getDataRange().getValues();
    for(let i=1;i<rows.length;i++) {
      if(rows[i][0]===d.orderId) {
        SS.getSheetByName('Orders').getRange(i+1,8).setValue(d.status); break;
      }
    }
  }
  return ContentService.createTextOutput('OK');
}
function doGet(e) {
  if(e.parameter.action==='getOrder') {
    const rows=SS.getSheetByName('Orders').getDataRange().getValues();
    for(let i=1;i<rows.length;i++) {
      if(rows[i][0]===e.parameter.orderId)
        return ContentService.createTextOutput(JSON.stringify({
          orderId:rows[i][0],customer:{name:rows[i][1]},status:rows[i][7]
        })).setMimeType(ContentService.MimeType.JSON);
    }
  }
  return ContentService.createTextOutput('{}').setMimeType(ContentService.MimeType.JSON);
}

Step 3 — Deploy on GitHub Pages (Free Hosting)

  1. Create account on github.com
  2. New Repository → Name: ambeypooja → Public
  3. Upload all 3 files: index.html, admin.html, db.js
  4. Settings → Pages → Source: main branch → Save
  5. Your store is live at https://yourusername.github.io/ambeypooja 🎉
✅ Total monthly cost: ₹0. Only Razorpay 2% per paid order.
